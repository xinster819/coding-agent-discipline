# 三指标台账 —— 回答"这套体系有没有效"（用数据，不用感觉）

> 采集规则：每次 session 收工（handoff-save / retro）时回填当周行；用户也可随手补。
> - **用户纠正次数**：用户指出 agent 错误/不满意的次数（任何工具任何项目）
> - **同类复发**：能指向已有失败卡（evals/cases/#N）的错误——记卡号
> - **新变体首现**：立了新失败卡的错误
> - **正例**：立了正例卡（exemplars）的漂亮操作——也要数，别只数错
>
> **判定（基线两周后）**：纠正次数与同类复发下降 = 体系有效（新变体不降可接受——规则天然滞后一步）；全都不降 = 按 roadmap「诚实边界」换方向，不恋战。
